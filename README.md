# 주머니 속 여행 가이드

## 1. Project Abstract
  
### 프로젝트 개요

* 목적
    * LLM 기반 울산광역시 관광지 어드바이저 챗봇 설계

* 주요 기능
    * 정확한 관광지 정보 제공
    * 챗봇 서비스를 통한 사용자 대화 기반 추천
    * 편의 서비스 제공 (네이버 번역 API를 활용한 번역 서비스 등)
