## 1. Project Abstract

- ICT 비지니스 경진대회 본선 진출 (Ongoing)
- 울산공공데이터 창업 경진대회 제품개발 부문 본선 진출 (Ongoing)
  
### 프로젝트 개요

* 목적
    * LLM 기반 울산광역시 관광지 어드바이저 챗봇 설계

* 주요 기능
    * 정확한 관광지 정보 제공
    * 챗봇 서비스를 통한 사용자 대화 기반 추천
    * 편의 서비스 제공 (네이버 번역 API를 활용한 번역 서비스 등)
